# ansible-workspace
This projects aim to be an automatic installer for 
all my custom settings.

## Installation 
### OpenSUSE
To install on Opensuse run:
```
bash <(curl -fsSL https://github.com/rgiovanardi/ansible-workspace/raw/main/bin/install.opensuse.sh)
```
